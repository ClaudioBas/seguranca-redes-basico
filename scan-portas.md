# Scan de Portas – Host Local

Este teste teve como objetivo identificar portas abertas
no próprio equipamento utilizado nos testes.

## Ferramenta
- Nmap

## Comando utilizado
```bash
nmap 127.0.0.1

Resultado
Foram identificadas portas abertas relacionadas a serviços
do sistema operacional local.

Considerações de Segurança
Serviços em execução localmente devem ser monitorados e
restritos sempre que possível para reduzir a superfície de ataque.
