# Segurança de Redes – Projeto Básico

Projeto prático de Segurança da Informação focado em
descoberta de rede, scan de portas e análise básica de riscos
utilizando Nmap.

## Ferramentas
- Nmap
- Windows

## Objetivo
Demonstrar conhecimentos iniciais em redes e segurança da informação
aplicados a um ambiente real e controlado.

## Observação
Os testes foram realizados apenas em ambiente próprio,
sem violar políticas ou leis.
